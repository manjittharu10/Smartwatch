# Smartwatch
A simple and interactive Smart Watch UI built using HTML, CSS, and JavaScript. This project simulates basic smart watch functionalities such as displaying real-time time, date, and interactive UI elements, focusing on clean design and smooth user experience.

 Features:

Real-time digital clock

Dynamic date display

Interactive buttons / UI controls

Clean and responsive smart watch interface

Smooth JavaScript-based updates

 Tech Stack:

HTML

CSS

JavaScript (Vanilla JS)

What I Learned:

Working with JavaScript Date object

Real-time UI updates

DOM manipulation

Designing interactive frontend components

Writing clean and readable JavaScript code

This project helped me understand how real-world UI components can be simulated using core web technologies and strengthened my frontend fundamentals.
